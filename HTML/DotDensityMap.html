Section 4: Maps

Dot Density Choropleth

After trying many maps implementations (like d3.js) I decided to use MapBoxGL public API, since it is the one with the best performance, considering the scale of our dataset. Also, it provides a simple way to cluster points, which in this case is a very useful feature, because we're plotting a lot of points and this way we can visualize the distribution of the trees around the country. Even that, I couldn't find any reliable way of plotting every tree on the dataset (there are only 3,000,000 included in this map).
It is important to mention that all the points could have been displayed, but by fetching the data as a GeoJSON file, but this was not possible due the limitations of file sizes of GitHub pages. Take on consideration that the points take some seconds to load on the map.
